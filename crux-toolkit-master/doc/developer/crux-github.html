Managing the Crux GitHub Project

We use GitHub to host the Crux Git repository and the Crux website. Kaipo Tamura, Charles Grant, and Bill Noble are the maintainers for the project. Contact one of them to be added as a developer on the project. We have a utility GitHub account with the user name cruxtoolkit. The utility account is used by our TeamCity server to upload files to the web site. Contact Charles for the password for this account.
